Inertial Navigation System (INS) Simulation using Python

This project presents a simulation of an Inertial Navigation System (INS) developed using accelerometer and gyroscope data. The system estimates motion parameters such as velocity, position, trajectory and orientation through numerical integration and basic sensor fusion techniques.

Project Objectives

Simulate the working principle of inertial navigation used in aerospace systems
Estimate motion parameters from IMU sensor data
Understand the impact of sensor noise and drift
Apply filtering and smoothing for improved signal accuracy
Demonstrate basic sensor fusion of accelerometer and gyroscope data

Methodology

Collected IMU sensor data (accelerometer and gyroscope)
Applied noise modelling to replicate real-world sensor behaviour
Implemented smoothing techniques to reduce signal fluctuations
Estimated velocity and position using numerical integration
Computed orientation using gyroscope data
Performed basic sensor fusion for improved motion estimation

Technologies Used

Python
NumPy
Pandas
Matplotlib

Key Outputs

The system successfully estimates:
Velocity from acceleration data
Position through integration
Orientation from gyroscope signals
Trajectory of motion
Fused motion estimation combining multiple sensors

Relevance

Inertial Navigation Systems are widely used in:
Launch vehicles
Spacecraft guidance systems
Autonomous navigation
Aerospace sensor systems

This project demonstrates the fundamental concepts behind inertial sensing and navigation used in real-world aerospace applications.
